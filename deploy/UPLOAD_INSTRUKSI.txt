==========================================
CARA UPLOAD VIA FILEZILLA KE CPANEL
==========================================

LANGKAH 1: BUKA FILEZILLA
   - Connect ke server cPanel Anda dengan credentials FTP/SFTP
   - Host: ftp.solusicodekata.com atau IP server
   - Username: username cPanel Anda
   - Password: password cPanel Anda
   - Port: 21 (FTP) atau 22 (SFTP)

LANGKAH 2: NAVIGASI KE FOLDER SUBDOMAIN
   - Di FileZilla, navigasi ke: /public_html/monitoring/
   - Pastikan folder ini ada (atau buat jika belum ada)
   - Ini adalah folder root untuk subdomain Anda

LANGKAH 3: UPLOAD SEMUA ISI FOLDER DEPLOY/
   ‚úÖ Upload folder server/ ‚Üí menjadi server/
   ‚úÖ Upload folder public/ ‚Üí menjadi public/
   ‚úÖ Upload file package.json ‚Üí ke root folder
   ‚úÖ (Opsional) Upload ENV_TEMPLATE.txt sebagai referensi

   ‚ö†Ô∏è PASTIKAN: Upload isi folder deploy/, bukan folder deploy/ itu sendiri
   Struktur setelah upload:
   /public_html/monitoring/
   ‚îú‚îÄ‚îÄ server/

LANGKAH 4: BUAT FILE .env DI ROOT FOLDER SERVER
   - Di File Manager cPanel (atau FileZilla), buka /public_html/monitoring/
   - Buat file baru bernama: .env
   - Edit file .env dan isi dengan:
     
     DB_HOST=localhost
     DB_USER=solz1468_solkit
     DB_PASSWORD=DemiAllah@1
     DB_NAME=solz1468_monitoring
     PORT=5000
     NODE_ENV=production

   ‚ö†Ô∏è PORT=5000 akan di-update nanti setelah create Node.js App

LANGKAH 5: SETUP NODE.JS DI CPANEL (Lihat SETUP_CPANEL_LANGKAH.txt)
   - Buka Node.js Selector di cPanel
   - Klik "CREATE APPLICATION"
   - Isi form:
     * Node.js version: Pilih versi terbaru (18.x atau 20.x)
     * Application mode: Production ‚úÖ
     * Application root: /public_html/monitoring
     * Application URL: monitoring.solusicodekata.com
     * Application startup file: server/index.js
   - Klik "CREATE"
   - CATAT PORT yang diberikan cPanel

LANGKAH 6: UPDATE FILE .env DENGAN PORT YANG BENAR
   - Edit file .env di server
   - Ganti PORT=5000 dengan PORT dari cPanel (misal: PORT=5001)

LANGKAH 7: TAMBAHKAN ENVIRONMENT VARIABLES (RECOMMENDED)
   - Di halaman Node.js App yang sudah dibuat
   - Klik "+ ADD VARIABLE" di bagian Environment variables
   - Tambahkan semua variabel dari file .env
   - Ini lebih prioritas daripada file .env

LANGKAH 8: INSTALL DEPENDENCIES
   - Via Node.js Selector: Klik "NPM Install" atau "Run NPM Install"
   - ATAU via Terminal SSH: cd /public_html/monitoring && npm install

LANGKAH 9: RESTART APLIKASI
   - Via Node.js Selector: Klik "Restart App" atau "Reload App"

LANGKAH 10: TEST
   - Buka browser: https://monitoring.solusicodekata.com
   - Test semua fitur aplikasi
