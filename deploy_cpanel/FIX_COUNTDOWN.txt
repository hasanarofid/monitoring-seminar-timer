CARA UPDATE:
===========================================

LANGKAH 1: Rebuild React App
-----------------------------
1. Di komputer lokal, masuk ke folder client:
   cd client

2. Rebuild aplikasi React:
   npm run build

3. File build akan tersimpan di folder client/build/


LANGKAH 2: Update File di Server
----------------------------------
1. Login ke cPanel
2. Buka File Manager
3. Navigasi ke folder subdomain monitoring.solusicodekata.com

4. Upload file berikut:
   
   A. Upload semua isi client/build/ ke root folder subdomain
      - Ganti file index.html
      - Ganti folder static/ (semua isinya)
   
   B. Upload file api/index.php yang sudah diperbaiki
      - Ganti file api/index.php di server
      (File ini sudah ada di folder deploy_cpanel/api/index.php)


LANGKAH 3: Clear Browser Cache
-------------------------------
1. Clear browser cache (Ctrl+Shift+Delete)
2. Atau gunakan Incognito/Private mode
3. Atau hard refresh (Ctrl+F5)


LANGKAH 4: Test
---------------
1. Test API:
   https://monitoring.solusicodekata.com/api/seminars/active
   
   Seharusnya waktu_berjalan sudah dihitung dengan benar

2. Test Aplikasi:
   https://monitoring.solusicodekata.com
   
   Seharusnya countdown waktu sudah benar

===========================================
ATAU UPLOAD ULANG SELURUH FOLDER deploy_cpanel
===========================================

Jika lebih mudah, Anda bisa upload ulang seluruh folder deploy_cpanel:

1. Rebuild React app (npm run build di folder client/)
2. Copy semua isi client/build/ ke deploy_cpanel/
3. Upload ulang seluruh isi deploy_cpanel/ ke server
4. Clear browser cache
5. Test aplikasi
