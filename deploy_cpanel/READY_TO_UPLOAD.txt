CARA UPLOAD KE SERVER:
===========================================

LANGKAH 1: Login ke cPanel
---------------------------
1. Login ke cPanel Anda
2. Buka File Manager

LANGKAH 2: Navigasi ke Folder Subdomain
----------------------------------------
1. Navigasi ke folder subdomain monitoring.solusicodekata.com
   (Biasanya: public_html/monitoring/ atau monitoring/)

LANGKAH 3: Upload Semua File
-----------------------------
1. Upload SEMUA isi folder deploy_cpanel/ ke root folder subdomain
   
   File yang perlu di-upload:
   - ✅ Folder api/ (dengan semua isinya)
   - ✅ Folder static/ (dengan semua isinya)
   - ✅ File index.html
   - ✅ File asset-manifest.json
   - ✅ File .htaccess
   - ✅ File brigthon.jpeg (jika ada)

2. Pastikan struktur folder sama seperti di deploy_cpanel/

LANGKAH 4: Set Permission File
------------------------------
Set permission file:
- File PHP (.php): 644
- File .htaccess: 644
- File HTML/CSS/JS: 644
- Folder (api/, static/): 755

LANGKAH 5: Clear Browser Cache
------------------------------
1. Clear browser cache (Ctrl+Shift+Delete)
2. Atau gunakan Incognito/Private mode
3. Atau hard refresh (Ctrl+F5)

LANGKAH 6: Test Aplikasi
-------------------------
1. Test API:
   https://monitoring.solusicodekata.com/api/seminars
   
   Seharusnya muncul JSON response

2. Test Aplikasi:
   https://monitoring.solusicodekata.com
   
   Seharusnya:
   - ✅ Tidak ada error CORS
   - ✅ Aplikasi bisa load data
   - ✅ Countdown waktu benar
   - ✅ Tidak berkedip
   - ✅ Resume dari jeda berfungsi
